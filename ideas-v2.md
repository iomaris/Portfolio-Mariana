# Redesign - Portfólio de Mariana (V2)

## Design Philosophy: Sofisticação Moderna com Paleta Rosa

### Paleta de Cores Refinada
- **Rosa Primário**: #C97B8F (rosa escuro mas clarinho - sofisticado)
- **Rosa Claro**: #E8C5D0 (para backgrounds e destaques)
- **Branco**: #FEFBF9 (off-white quente)
- **Cinza Escuro**: #1F1F1F (para texto principal)
- **Cinza Médio**: #8B8B8B (para texto secundário)
- **Ouro Sutil**: #D4AF8F (para acentos e detalhes)

### Tipografia Moderna
- **Display/Títulos**: Poppins Bold ou Montserrat Bold (moderno, geométrico)
- **Subtítulos**: Poppins SemiBold
- **Body**: Poppins Regular (moderno, limpo, muito legível)
- **Proporção**: Títulos grandes (3-4rem), subtítulos (1.5-2rem), body (1rem)

### Layout Elaborado
1. **Hero Section**: 
   - Assimétrico com imagem de fundo sofisticada
   - Gradiente rosa sutil sobreposto
   - Texto em posição estratégica com animações de entrada

2. **Sobre Mim**:
   - Grid assimétrico 2-3 colunas
   - Card com foto circular (com borda rosa)
   - Texto com linhas decorativas
   - Badges com interesses (hover effects)

3. **Projetos**:
   - Cards com hover 3D
   - Imagens com overlay gradiente rosa
   - Botões com efeito de expansão
   - Timeline visual para projetos

4. **Interesses**:
   - Grid com cards que se sobrepõem
   - Ícones grandes e detalhados
   - Imagens com bordas rosa
   - Animações ao scroll

5. **Contato**:
   - Formulário elegante com inputs customizados
   - Ícones com animações
   - Background com padrão sutil

### Elementos Visuais Detalhados
- Linhas decorativas em rosa
- Dividers com efeito de onda
- Sombras suaves (rosa-tinted)
- Bordas rosa em cards e imagens
- Gradientes rosa para backgrounds
- Animações fluidas e sofisticadas

### Interações
- Hover effects em todos os elementos interativos
- Scroll animations (fade-in, slide-in)
- Transições suaves (300-500ms)
- Efeitos de parallax sutis
- Botões com ripple effect
